# tcp-review
This code is inspired by [this excellent YouTube video](https://www.youtube.com/watch?v=F27PLin3TV0)

## System Requirements
A Unix-like OS

## Compiling
In your terminal run make:
```bash
make
```

## Running the server
In your terminal:
```bash
./server
```

## Running the client
In your terminal:
```bash
./client [address of server]
```
